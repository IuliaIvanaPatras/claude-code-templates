# Skills

Skills are reusable prompts that teach Claude specific patterns for frontend development with React 19.2, Next.js 16, TypeScript 6, and Tailwind CSS v4.

## Available Skills

| Skill | Argument Hint | Description |
|-------|---------------|-------------|
| [react-nextjs](react-nextjs/) | — | React 19.2 + Next.js 16 — Server Components, Cache Components, proxy.ts, View Transitions |
| [code-quality](code-quality/) | `[file-or-directory]` | TypeScript/React code review — clean code, type safety, accessibility checks |
| [design-patterns](design-patterns/) | `[pattern-name]` | Frontend patterns — Composition, Compound Components, Custom Hooks |
| [performance-patterns](performance-patterns/) | `[page-or-component]` | Core Web Vitals, Turbopack optimization, bundle splitting, Cache Components |
| [accessibility-patterns](accessibility-patterns/) | `[component-or-page]` | WCAG 2.2, ARIA patterns, keyboard navigation, screen readers |
